# Claude Code

Anthropic's official CLI tool for AI-assisted software engineering. Runs in the terminal and provides an interactive agent that helps with coding tasks.

## Installation characteristics

- **Native installer** (recommended): Zero dependencies, auto-updates. Installed via curl/PowerShell one-liner.
- **Deprecated npm method**: Requires Node.js 18+.
- **Windows prerequisite**: Git for Windows is required.
- **Account required**: Pro, Max, Teams, Enterprise, or Console plan. Free plan does not include Claude Code.
- **Authentication**: Browser-based OAuth flow after first `claude` command.

## Key paths

- Binary: `~/.local/bin/claude` (macOS/Linux), `%USERPROFILE%\.local\bin\claude.exe` (Windows)
- Config: `~/.claude/`
