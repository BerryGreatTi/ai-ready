# AIReady

The project name and brand for this cross-platform installer helper. AIReady automates the installation and onboarding of AI coding tools (Claude Code, OpenClaw) for beginners.

The name conveys: "After running this tool, your computer is AI Ready."

## File naming convention

Release artifacts follow the pattern: `AIReady-{TargetTool}-{Platform}.{ext}`

Examples:
- `AIReady-ClaudeCode-Win.exe`
- `AIReady-OpenClaw-Mac.app`
- `AIReady-ClaudeCode-Linux.sh`
