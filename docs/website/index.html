macOS: Gatekeeper bypass instructions

Because AIReady is distributed outside the Mac App Store, macOS Gatekeeper may block it from running. Follow these steps to allow it:

  1. Open System Settings (or System Preferences on older macOS).
  2. Go to Privacy & Security.
  3. Scroll to the Security section and click Open Anyway next to the AIReady message.
  4. Alternatively, right-click the app in Finder and choose Open, then confirm.

Or run this command in Terminal after downloading:

xattr -d com.apple.quarantine /path/to/AIReady-ClaudeCode-Mac.app
Verify checksums

SHA-256 checksums are published with each release to verify file integrity.

Download CHECKSUMS.txt
